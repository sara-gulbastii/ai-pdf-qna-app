# ai-pdf-qna-app# PDF RAG Q&A Chatbot Web App

A full-stack AI web app that lets users upload PDFs and ask questions — answers are pulled accurately from document content using Retrieval-Augmented Generation (RAG).

## Features
- Upload multiple PDFs
- Chat interface for natural questions
- Powered by LangChain, OpenAI GPT-3.5, and Chroma vector DB
- No hallucinations — answers only from your documents
